# Notebook structure

## Title
Markdown cell with the project name (e.g. "Project 01", "Project 02", etc.) as `H1` and the project name (e.g. "Energy-Delay Optimization of a Three-Stage CMOS Buffer", etc.) directly below as `simple text`.

## Imports
Code cell with all the necessary libraries imported.

## Constants
Code cell with all the necessary constants defined.

## Project initialization
Code cell with all the necessary initializations for the project.

## Steps
### Step title
Markdown cell with the step name (e.g. "STEP 1: Minimum inverter sizing", etc.) as `H2` and the step description directly below as `simple text` using code brackets for numeric values (e.g. `10nm`, `100nm`, etc.), latex equations and **bold** and _italic_ text format.

### Code
A series of code cell subdivided for each sub-step of the step.
